Sobre a Aplicação

Esta aplicação é uma demonstração de integração com a API Pokémon. Ela permite visualizar uma lista de Pokémon e exibir detalhes de cada um deles.

A API Pokémon fornece dados detalhados sobre Pokémon, incluindo informações como nome, habilidades, tipos e muito mais.

Esta aplicação utiliza Angular para o frontend e consome os dados da API Pokémon para exibir as informações.

Para mais informações sobre a API Pokémon, visite o site oficial da PokéAPI.
